O desenvolvimento do gerenciador de geração de images foi feita com auxílio da biblioteca gráfica
openGL, que é um ambiente de desenvolvimento para aplicações gráficas 2D e 3D e especifica uma
interface de software padrão para hardware de processamento gráfico.

Chamaremos de vértices os pares ordenados
$$
    \(x, y\) \in \mathbb{R}
$$

Para desenhar uma imagem na tela usando o openGL, é necessário especificar o tipo de entidade
geométrica, isto é, pontos, segmentos de linha ou triângulos que consistem de vértices. As
entidades geométricas
