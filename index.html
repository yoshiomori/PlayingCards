Descrição do Trabalho

Pretendemos neste projeto desenvolver um aplicativo para dispositivos móveis com S.O Android que simule da "melhor maneira possível" um baralho para jogos entre amigos, de modo que possa ser usado em qualquer jogo local como um substituto de um baralho real.

Iremos também definir uma linguagem para especificar jogos de baralho para ser utilizado pelo aplicativo. Dessa forma, um usuário seria capaz de definir seu próprio jogo e agilizar o embaralhar e o processo de distribuição das cartas para os outros jogadores.
