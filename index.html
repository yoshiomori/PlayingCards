Descrição do Trabalho

Pretendemos neste projeto desenvolver um aplicativo para dispositivos móveis com S.O Android que simule da "melhor maneira possível" um baralho para jogos entre amigos, de modo a ser usado em qualquer jogo local como um substituto de um baralho real.

Iremos também definir uma linguagem para especificar jogos de baralho para ser utilizado pelo aplicativo. Dessa forma, um usuário seria capaz de definir seu próprio jogo e agilizar o embaralhar e o processo de distribuição das cartas para os outros jogadores.

Nesse projeto, entendemos como a "melhor maneira possível" de simular um baralho como a maneira que conserva as seguintes características:
  • Cada carta pode ser virada em grupo ou sozinhas, tanto para mostrar seu valor quanto para esconder com um duplo toque na tela sobre o baralho
  • Cada carta pode ser passada entre os jogadores e pode ser sacada do monte, em grupo ou sozinhas, com o arrastar dos dedos na tela na direção do dispositivo do outro jogador

Tarefas

  • Comunicação entre os dispositivos
  • Detecção da direção de um dispositivo para outro
  • Abstração de uma linguagem para especificar jogos de baralho
  • Modelo
  • Visual
  • Controle
